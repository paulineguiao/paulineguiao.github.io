HTML Cheat Sheet

Tag Description
<html></html> Root of an HTML document. Begins and ends the HTML document. Everything else will be inside this tag.
<head></head> Contains metadata. The documents header, contains various information that does not appear in the body of the document.
<title></title> Page title. Typically displayed in the browser title bar.
<body></body> Main content, essentially what the user sees inside his or her browser window.
<h1></h1> Defines headings from largest (h1) to smallest (h6).
<p></p> Creates blocks of text.
<a></a> Creates a hyperlink that requires the "href" attribute that tells the link where to go.
<img> Image used for including images in the flow of text.
<ul></ul> Unordered list, typically rendered with bullet points in front of each list item.
<ol></ol> Ordered list, typically rendered with numbers in front of each list item.
<li></li> List item, represents a single item in a list. Must be nested inside <ul> or <ol>.
<table></table> Table container, organize data in rows and columns.
<tr></tr> Table row, which group cells horizontally.
<td></td> Table data cells, holds the actual content of the table.
<form></form> Form container, it wraps input elements to create a form.
